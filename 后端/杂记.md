# 软件常用版本号

| 版本标志                        | 含义       |
| ------------------------------- | ---------- |
| Release                         | 正式版本   |
| RC stands for Release Candidate | 后选版本   |
| M stands for milestone          | 里程碑版本 |
| Snapshot                        | 快照版本   |
